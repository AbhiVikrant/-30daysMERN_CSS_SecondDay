Introduction and Applying CSS to HTML Document

Types of Applying CSS:

  1. Internal CSS:

    Applying CSS using style tag inside head.

  2. Inline CSS:

    Applying CSS within tag using style

  3. External CSS:

    Applying CSS using embedding external file to document and mostly preferred way for writting CSS due to ease to maintain.



Types of Selectors in CSS:

  • Universal Selector:

    It applied to whole body elements and denoted as '*'.

  • Element Selectors:

    This is the method of applying css by directly on element using tag name. And if same properties to be applied to more than one element, it can be grouped together using , in between them.

  • Class Selectors:

    It is as providing class name to the element and using '.' with className, we can apply css. More than one element can have a single class.

  • ID selectors:

    Unique and used usin '#' with ID_Name.
