Introduction and Applying CSS to HTML Document

Types of Applying CSS:

  1. Internal CSS:

    Applying CSS using style tag inside head.

  2. Inline CSS:

    Applying CSS within tag using style

  3. External CSS:

    Applying CSS using embedding external file to document and mostly preferred way for writting CSS due to ease to maintain.



Types of Selectors in CSS:

  • Universal Selector:

    It applied to whole body elements and denoted as '*'.

  • Element Selectors:

    This is the method of applying css by directly on element using tag name. And if same properties to be applied to more than one element, it can be grouped together using , in between them.

  • Class Selectors:

    It is as providing class name to the element and using '.' with className, we can apply css. More than one element can have a single class.

  • ID selectors:

    Unique and used usin '#' with ID_Name.

Bacground properies allows us to set up background image to either an element or body.Its constituent properies with initial values as:

  • background-image as provided url(...location) and its initial value as none.
  • background-position sets the initial position of the image. initial value as 0% 0%.
  • background-size specifies the size of the background image. It may be cover, contain or auto or in percentages.
  • background-repeat stes how background images are repeated. Its value may be repeat, repeat-x or y, space, round or space, repeat means combination of two or no repeat.
  • background-origin sets the background origin from border starts, insidethe border or inside the padding. Its default value as padding- box and other values as border-box or content-box.
  • background-clip determines whether background extends underneath its border-box, padding-box or content-box. Default value is border-box and text be another value also.
  • background-attachment sets position according to viewport whether as fixed,scroll or local. Default value is scroll.
